# 🏏 IPL Analytics Dashboard 2008–2025

An interactive data analytics dashboard built on **18 seasons of IPL data** — covering 1,169 matches and 278,205 ball-by-ball deliveries.

---

## ✨ Features

- **Home & Overview** — KPI cards, key findings, IPL champions timeline
- **Team Analysis** — all-time win records, season heatmap, season trends
- **Player Analysis** — top batsmen, bowlers, strike rates, economy rates, POTM awards
- **Match Insights** — toss analysis, top venues, phase-wise runs & wickets, dismissal types
- **Head-to-Head** — win share, match history, season-wise results between any two teams
- **Details** — deep profile for any player (runs, avg, SR, wickets, economy) or team (titles, win%, season record)
- **Live Predictor** — fetches live IPL scores from Cricbuzz and predicts win probability using current match state + historical win rates

---

## 🛠️ Tech Stack

`Python` · `Streamlit` · `pandas` · `Matplotlib` · `Seaborn` · `Cricbuzz API`

---

## 🚀 Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
